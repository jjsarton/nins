setup_wniqd.exe

This file what created with pelles C on a 32 bit system and shall run on Windows XP SP3 and later. Compilation what made on a Linux system with wine.

The source are for Windows XP. Windows Vista and Windows 7 may contain a better IPv6 support but I don't own such a system but the code shall run.

wniqd is installed as service and include the 32 bit version of nssm.exe, see: http://nssm.cc/

If you want to deinstall wniqd, go to the installtion directory and run uninst.exe.
